# Clickbait Detection using DistilBERT

Fine-tuned DistilBERT on clickbait headline dataset.  
Achieved 98.9% validation accuracy.

## How to run
- Open the .ipynb in Google Colab
- Runtime → Run All

## Tech used
- Python, HuggingFace Transformers, PyTorch
- DistilBERT fine-tuned for sequence classification
